# Product Templates

Reusable document templates for software product planning and execution.

## Structure

```
product/
├── discovery/
│   ├── Project-Brief-Template.md
│   ├── PRD-Template.md
│   ├── Technical-Spec-Template.md
│   ├── Persona-Template.md
│   ├── Experience-Canvas-Template.md
│   └── Product-Roadmap-Template.md
├── execution/
│   ├── Project-Plan-Template.md
│   ├── Sprint-Plan-Template.md
│   ├── Status-Update-Template.md
│   ├── Goals-Signals-Measures-Template.md
│   └── Trade-Off-Sliders-Template.md
├── launch/
│   ├── Release-Notes-Template.md
│   ├── Postmortem-Template.md
│   └── Retrospective-Template.md
└── ongoing/
    ├── DACI-Decision-Template.md
    ├── Meeting-Notes-Template.md
    └── Team-Health-Monitor-Template.md
```

## Templates

### Discovery
| Template | Purpose |
|----------|---------|
| [Project Brief](discovery/Project-Brief-Template.md) | Initial pitch: problem statement, target audience, goals, stakeholders |
| [PRD](discovery/PRD-Template.md) | Product requirements: user stories, features, acceptance criteria |
| [Technical Spec](discovery/Technical-Spec-Template.md) | Architecture, API design, data models, deployment strategy |
| [Persona](discovery/Persona-Template.md) | User profiles: demographics, goals, pain points, behaviors |
| [Experience Canvas](discovery/Experience-Canvas-Template.md) | User journey mapping: goals, pain points, value proposition |
| [Product Roadmap](discovery/Product-Roadmap-Template.md) | Release timeline: milestones, initiatives, dependencies |

### Execution
| Template | Purpose |
|----------|---------|
| [Project Plan](execution/Project-Plan-Template.md) | Milestones, tasks, dependencies, risks, progress tracking |
| [Sprint Plan](execution/Sprint-Plan-Template.md) | Agile sprint planning: backlog, capacity, ceremonies, retro |
| [Status Update](execution/Status-Update-Template.md) | Regular reporting: progress, blockers, metrics, next steps |
| [Goals, Signals, Measures](execution/Goals-Signals-Measures-Template.md) | Success metrics: baselines, targets, measurement methods |
| [Trade-off Sliders](execution/Trade-Off-Sliders-Template.md) | Priority visualization: scope vs. time, quality vs. speed |

### Launch
| Template | Purpose |
|----------|---------|
| [Release Notes](launch/Release-Notes-Template.md) | Changelog: features, fixes, breaking changes, upgrade instructions |
| [Postmortem](launch/Postmortem-Template.md) | Incident review: timeline, root cause, lessons learned |
| [Retrospective](launch/Retrospective-Template.md) | Process improvement: 4Ls (Liked, Learned, Lacked, Longed For) |

### Ongoing
| Template | Purpose |
|----------|---------|
| [DACI Decision](ongoing/DACI-Decision-Template.md) | Structured decisions: Driver, Approver, Contributors, Informed |
| [Meeting Notes](ongoing/Meeting-Notes-Template.md) | Agenda, discussion, decisions, action items |
| [Team Health Monitor](ongoing/Team-Health-Monitor-Template.md) | Team assessment: 10 dimensions of team effectiveness |

## Usage

1. Copy the relevant template to your product folder
2. Rename it (drop the `-Template` suffix)
3. Fill in the sections
