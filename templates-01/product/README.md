# Templates

Reusable document templates for software project planning and execution.

## Structure

```
templates/
├── discovery/
│   ├── Project_Brief_Template.md
│   ├── PRD_Template.md
│   ├── Technical_Spec_Template.md
│   ├── Persona_Template.md
│   ├── Experience_Canvas_Template.md
│   └── Product_Roadmap_Template.md
├── execution/
│   ├── Project_Plan_Template.md
│   ├── Sprint_Plan_Template.md
│   ├── Status_Update_Template.md
│   ├── Goals_Signals_Measures_Template.md
│   └── Trade_Off_Sliders_Template.md
├── launch/
│   ├── Release_Notes_Template.md
│   ├── Postmortem_Template.md
│   └── Retrospective_Template.md
└── ongoing/
    ├── DACI_Decision_Template.md
    ├── Meeting_Notes_Template.md
    └── Team_Health_Monitor_Template.md
```

## Templates

### Discovery Phase
| Template | Purpose |
|----------|---------|
| [Project Brief](discovery/Project_Brief_Template.md) | Initial pitch: problem statement, target audience, goals, stakeholders |
| [PRD](discovery/PRD_Template.md) | Product requirements: user stories, features, acceptance criteria |
| [Technical Spec](discovery/Technical_Spec_Template.md) | Architecture, API design, data models, deployment strategy |
| [Persona](discovery/Persona_Template.md) | User profiles: demographics, goals, pain points, behaviors |
| [Experience Canvas](discovery/Experience_Canvas_Template.md) | User journey mapping: goals, pain points, value proposition |
| [Product Roadmap](discovery/Product_Roadmap_Template.md) | Release timeline: milestones, initiatives, dependencies |

### Execution Phase
| Template | Purpose |
|----------|---------|
| [Project Plan](execution/Project_Plan_Template.md) | Milestones, tasks, dependencies, risks, progress tracking |
| [Sprint Plan](execution/Sprint_Plan_Template.md) | Agile sprint planning: backlog, capacity, ceremonies, retro |
| [Status Update](execution/Status_Update_Template.md) | Regular reporting: progress, blockers, metrics, next steps |
| [Goals, Signals, Measures](execution/Goals_Signals_Measures_Template.md) | Success metrics: baselines, targets, measurement methods |
| [Trade-off Sliders](execution/Trade_Off_Sliders_Template.md) | Priority visualization: scope vs. time, quality vs. speed |

### Launch Phase
| Template | Purpose |
|----------|---------|
| [Release Notes](launch/Release_Notes_Template.md) | Changelog: features, fixes, breaking changes, upgrade instructions |
| [Postmortem](launch/Postmortem_Template.md) | Incident review: timeline, root cause, lessons learned |
| [Retrospective](launch/Retrospective_Template.md) | Process improvement: 4Ls (Liked, Learned, Lacked, Longed For) |

### Ongoing
| Template | Purpose |
|----------|---------|
| [DACI Decision](ongoing/DACI_Decision_Template.md) | Structured decisions: Driver, Approver, Contributors, Informed |
| [Meeting Notes](ongoing/Meeting_Notes_Template.md) | Agenda, discussion, decisions, action items |
| [Team Health Monitor](ongoing/Team_Health_Monitor_Template.md) | Team assessment: 10 dimensions of team effectiveness |

## Usage

1. Copy the relevant template to your product folder
2. Rename it (remove `_Template` suffix)
3. Fill in the sections
